# BANKSYS - SISTEMA BANCÁRIO SIMPLES
Projeto acadêmico desenvolvido em Java para exercitar os conceitos fundamentais de POO (Programação Orientada a Objetos).

## FUNCIONALIDADES:
Simula operações bancárias básicas.
***Criação de contas** com número e o saldo inicial.
***Realização de depósitos** (creditar).
***Realização de saques** (debitar).
***Transferências** para contas distintas.
***Consulta de saldo**.

## TECNOLOGIAS UTILIZADAS:
***Linguagem Java**.
***IDE:** Eclipse

## O QUE EU APRENDI COM ESSE PROJETO?
***Encapsulamento de dados** (uso de modificadores como o 'private').
***Lógica de Validação**.
***Gerenciamento de objetos** em coleções.
